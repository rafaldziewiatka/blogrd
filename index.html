Git początki

Jeśli trafiłeś na tego bloga, to zapewne chciałbyś nauczyć się korzystać z języka GIT, a ja postaram Ci się w tym pomóc. Pierwsze co musisz zrobić, to ściągnąć oprogramowanie ze strony https://git-scm.com/ i następnie zainstalować orogramowanie w wybranej lokalizacji. Proces instalacji nie jest trudny i nie jest od nas wymagane zaznaczanie jakichkolwiek opcji, więć śmiało możesz klikać "dalej". Po procesie instalacji utwórz nowy katalog, lub wybierz istniejący katalog roboczy. Kliknij prawym przyciskiem myszy i wybierz "Git Bash Here". Gdy wykonasz powyższą czynność otworzy się konsola. Na początek wpisz "pwd" i powinno pokazać Ci lokalizację wybranego katalogu. Żeby móc pracować z danym kataologiem powinieneś w pierwszej kolejności skonfigurować basha, a robi się to w sposób następujący: git config --global user.name "yourName" git config --global user.email "your@email.com" Powyższe komendy pozwalają podejrzeć Tobie oraz innym używającym basha, jakie zmiany wprowadzasz. Ostatnią komendą na dziś jest: git init Powyższa komenda powoduje w wybranej lokacji folderu o nazwie ".git", czyli inicjuje kontrolę treści w wybranym folderze.
